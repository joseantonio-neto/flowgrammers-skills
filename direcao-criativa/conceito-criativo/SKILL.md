# Conceito Criativo

Desenvolvimento do conceito central (big idea) e território criativo para campanhas e comunicações de marca no Brasil.

## Quando Usar
- Desenvolver conceito para nova campanha
- Criar big idea para lançamento de produto ou marca
- Revisar ou evoluciar conceito criativo existente

## Exemplos de Prompt
```
Nossa marca de tênis "Passo Firme" vai lançar uma linha para mulheres 35-50 anos
que voltam a se cuidar após anos priorizando a família. Budget: R$ 500k.
Crie o conceito criativo com: big idea em 1 frase, território emocional,
manifesto de 150 palavras e 3 execuções de exemplo (OOH, digital, social).
```

```
Preciso de um conceito criativo para a campanha de Black Friday de uma rede
de academia low cost. Fugir do clichê "desconto". Público: homens e mulheres
25-40 anos que querem mudar o estilo de vida. Proponha 3 conceitos diferentes.
```

## Saídas Típicas
- Big idea em 1 frase memorável
- Território emocional e posicionamento criativo
- Manifesto da campanha (100-200 palavras)
- 3 execuções de exemplo em canais diferentes
- Reason why criativo (por que este conceito funciona)
