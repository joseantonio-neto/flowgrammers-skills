# Naming

Criação de nomes para marcas, produtos e campanhas, com análise de disponibilidade para o mercado brasileiro: INPI, domínios .com.br e redes sociais.

## Quando Usar
- Criar nome para nova marca ou empresa
- Nomear produto, serviço ou linha de produtos
- Renomear marca existente (rebrand)

## Exemplos de Prompt
```
Preciso de 10 opções de nome para uma startup de saúde mental digital que conecta
psicólogos a pacientes via app. Público: adultos 25-45 anos nas capitais.
Para cada nome: origem/significado, memorabilidade, disponibilidade (.com.br, Instagram)
e fit com o setor de saúde brasileiro.
```

```
Vou lançar uma linha de bebidas funcionais com ingredientes da Amazônia.
Crie nomes que: evoquem origem brasileira, sejam pronunciáveis internacionalmente
e diferenciem no mercado. Gere 8 opções com análise de cada uma.
```

## Saídas Típicas
- Lista de opções de nome com análise individual
- Verificação de disponibilidade (.com.br, @instagram, INPI)
- Análise fonética e memorabilidade
- Avaliação de risco de confusão com marcas existentes
- Top 3 recomendações com justificativa estratégica
