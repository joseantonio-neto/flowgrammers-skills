# Direção de Arte

Briefing, referências e orientação para direção de arte em produções fotográficas, de vídeo e digitais para marcas brasileiras.

## Quando Usar
- Briefar diretor de arte ou fotógrafo para uma produção
- Definir linguagem visual de uma campanha
- Revisar referências antes de uma produção

## Exemplos de Prompt
```
Vou fazer uma produção fotográfica para lookbook de uma marca de roupas femininas
para executivas brasileiras 30-50 anos. Crie um documento de direção de arte com:
paleta de tons, estilo de iluminação, poses e expressões, cenários, styling
e 5 referências visuais descritivas.
```

```
Preciso briefar um diretor de vídeo para conteúdo de employer branding da minha
startup de tecnologia. Queremos mostrar o ambiente de trabalho de forma autêntica.
Crie o briefing com: mensagem, estilo (documental/lifestyle), casting, locações
e referências de marcas que fazem isso bem.
```

## Saídas Típicas
- Briefing de direção de arte estruturado
- Descrição da linguagem visual (luz, composição, cor, textura)
- Referências descritivas organizadas por elemento
- Checklist de produção (casting, locação, styling, equipamento)
- Moodboard textual com descrição de cada referência
