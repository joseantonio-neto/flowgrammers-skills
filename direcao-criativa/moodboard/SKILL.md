# Moodboard

Criação de moodboards textuais e curadoria de referências visuais para orientar produções fotográficas, de vídeo, design e campanhas.

## Quando Usar
- Briefar diretor de arte com referências visuais
- Alinhar expectativas visuais com cliente antes da produção
- Criar referências para equipe de design sem acesso a Pinterest/Behance

## Exemplos de Prompt
```
Crie um moodboard textual para uma campanha de perfume masculino premium brasileiro.
A marca se posiciona como: masculinidade contemporânea, sofisticação e raízes.
Descreva: paleta de tons, texturas, cenários, tipo de fotografia, styling masculino,
iluminação e 5 referências de marcas/campanhas que evocam este universo.
```

```
Preciso alinhar a direção visual de um e-commerce de decoração minimalista
com minha equipe. Crie um moodboard em texto com: paleta de cores (com nomes
descritivos), tipo de fotografia de produto, cenários, props e linguagem geral.
```

## Saídas Típicas
- Moodboard textual organizado por elemento visual
- Paleta de tons descrita com referências (não apenas hex)
- Descrição de texturas, materiais e superfícies
- Referências de marcas/campanhas por elemento
- Palavras de ancoragem do universo visual
