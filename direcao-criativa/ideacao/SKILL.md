# Ideação

Facilitação de sessões de brainstorming e ideação criativa usando técnicas estruturadas para gerar e selecionar ideias com times e projetos.

## Quando Usar
- Facilitar sessão de brainstorming com equipe
- Gerar volume de ideias para um desafio criativo
- Selecionar e priorizar ideias geradas

## Exemplos de Prompt
```
Preciso facilitar uma sessão de ideação de 2 horas com um time de 8 pessoas
para gerar ações de marketing para uma marca de cosméticos veganos no Carnaval.
Crie: roteiro da sessão com técnicas (SCAMPER, How Might We, etc.),
dinâmicas por bloco de tempo e critérios de votação das ideias.
```

```
Use a técnica dos 6 Chapéus de De Bono para analisar a ideia de lançar
um app de entregas ultrarrápidas (10 minutos) em cidades do interior BR.
Aplique cada chapéu e gere insights para cada perspectiva.
```

## Saídas Típicas
- Roteiro de sessão de ideação por bloco de tempo
- Técnicas criativas aplicadas ao desafio específico
- Volume de ideias geradas e organizadas por cluster
- Critérios de priorização das ideias
- Próximos passos para prototipar as top 3 ideias
