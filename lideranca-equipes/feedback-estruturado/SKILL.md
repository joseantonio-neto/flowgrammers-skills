# Feedback Estruturado

Frameworks e técnicas para dar e receber feedback de forma construtiva em equipes brasileiras: SBI, feedforward, conversas difíceis e PDI integrado.

## Quando Usar
- Preparar feedback para colaborador com problema de desempenho
- Estruturar conversa de reconhecimento e desenvolvimento
- Documentar feedback para registro de RH

## Exemplos de Prompt
```
Preciso dar feedback para meu analista de dados que produz trabalho excelente
mas não comunica riscos e atrasos proativamente. Ele deixou um relatório atrasar
2 dias sem avisar, afetando a reunião do board. Use o framework SBI para estruturar
o feedback e inclua um feedforward com comprometimentos concretos.
```

```
Tenho uma desenvolvedora que quer crescer para sênior mas ainda tem gaps em
visão de sistema e comunicação com stakeholders. Crie um roteiro de feedback
de desenvolvimento (não punitivo) com: pontos fortes, gaps com exemplos,
plano de desenvolvimento e próximos passos em 90 dias.
```

## Saídas Típicas
- Roteiro de feedback estruturado com SBI
- Feedforward com comprometimentos e prazo
- Documentação para registro de RH (se necessário)
- Preparação para possíveis reações defensivas
- Template de acompanhamento pós-feedback
