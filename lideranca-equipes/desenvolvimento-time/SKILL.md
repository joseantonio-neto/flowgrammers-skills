# Desenvolvimento de Time

Planos de Desenvolvimento Individual (PDI), trilhas de aprendizagem e frameworks de carreira para equipes em startups e PMEs brasileiras.

## Quando Usar
- Criar PDI personalizado para colaborador
- Estruturar trilha de desenvolvimento por área
- Montar framework de progressão de carreira

## Exemplos de Prompt
```
Minha desenvolvedora pleno quer crescer para sênior em 18 meses. Ela tem forte
habilidade técnica em backend Python mas gaps em: visão de sistema, mentoria
de juniors e comunicação técnica para stakeholders.
Crie um PDI com: competências, atividades práticas, recursos de aprendizagem,
marcos de avaliação bimestrais e critérios para promoção.
```

```
Preciso criar um framework de progressão de carreira para minha equipe de
6 designers. Quero ter: Junior, Pleno, Sênior e Lead com critérios claros
para cada nível, para acabar com a subjetividade nas promoções.
```

## Saídas Típicas
- PDI estruturado com competências, atividades e marcos
- Framework de progressão de carreira por área
- Recursos de desenvolvimento (cursos, livros, projetos, mentoria)
- Template de avaliação de desempenho por nível
- Critérios objetivos para promoção
