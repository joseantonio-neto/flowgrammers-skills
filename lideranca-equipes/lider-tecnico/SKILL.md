# Líder Técnico

Guia para a transição de especialista técnico (dev, designer, analista) para líder técnico ou manager, navegando os desafios únicos desta mudança no contexto de startups BR.

## Quando Usar
- Fazer a transição de dev/designer/analista para líder
- Apoiar especialista recém-promovido a líder
- Definir o papel de tech lead vs engineering manager

## Exemplos de Prompt
```
Fui promovido a tech lead de 6 devs há 2 meses. Ainda codifico 50% do tempo
e sinto culpa quando não estou no teclado. Meu time parece não me ver como
líder ainda. Crie um plano de transição de 90 dias para: reduzir coding gradualmente,
ganhar autoridade como líder e começar a ter impacto via pessoas, não código.
```

```
Qual a diferença prática entre tech lead e engineering manager no contexto de
startup brasileira com 20 pessoas? Crie uma comparação com: responsabilidades,
métricas de sucesso, habilidades necessárias e como cada papel escala.
```

## Saídas Típicas
- Plano de transição de 90 dias de especialista para líder
- Comparação tech lead vs engineering manager
- As 5 mudanças de mentalidade mais importantes na transição
- Framework de delegação gradual de tarefas técnicas
- Métricas de sucesso como líder técnico
