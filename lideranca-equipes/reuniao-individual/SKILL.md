# One-on-One (1:1)

Estrutura, perguntas e boas práticas para reuniões 1:1 eficazes que desenvolvem pessoas, aumentam o engajamento e constroem confiança.

## Quando Usar
- Estruturar rotina de 1:1s com a equipe
- Melhorar reuniões 1:1 que viraram só status report
- Preparar 1:1 difícil sobre performance ou carreira

## Exemplos de Prompt
```
Sou líder de uma equipe de 6 pessoas em startup e meus 1:1s de 30 minutos
estão virando update de sprint. Crie um template de 1:1 que: abra espaço
para o colaborador, discuta desenvolvimento, trate bloqueios e mantenha
a conexão humana. Inclua perguntas abertas para cada seção.
```

```
Preciso ter um 1:1 difícil com minha gerente de projetos sobre sua atitude
defensiva em reuniões com clientes. Como preparo essa conversa para que
ela receba bem e saiamos com um plano de ação claro?
```

## Saídas Típicas
- Template de 1:1 por duração (20min/30min/45min)
- Banco de perguntas abertas por tema (carreira, bloqueios, feedback)
- Roteiro para 1:1 difícil (performance, conflito, saída)
- Guia de escuta ativa para o líder durante o 1:1
- Formato de notas e acompanhamento pós-reunião
