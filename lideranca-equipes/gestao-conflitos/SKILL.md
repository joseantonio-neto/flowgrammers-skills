# Gestão de Conflitos

Abordagem estruturada para identificar, mediar e resolver conflitos em times brasileiros, preservando relacionamentos e a saúde do ambiente de trabalho.

## Quando Usar
- Mediar conflito entre dois membros da equipe
- Identificar e endereçar conflito latente no time
- Criar processo de resolução de conflitos para o time

## Exemplos de Prompt
```
Dois engenheiros do meu time estão em conflito sobre escolhas técnicas de arquitetura.
O conflito está afetando a colaboração do time todo. Crie um roteiro de mediação
em 3 reuniões: individual com cada um, conjunta para alinhamento e follow-up.
Inclua perguntas para cada etapa.
```

```
Minha equipe de marketing está dividida em dois grupos que não colaboram.
Acredito que a causa raiz é uma disputa por visibilidade com o CEO.
Como identifico se esta é realmente a causa e como redefino os papéis e
a estrutura para eliminar a competição interna disfuncional?
```

## Saídas Típicas
- Diagnóstico do tipo de conflito (interpessoal, estrutural, de processo)
- Roteiro de mediação passo a passo
- Perguntas-chave para cada parte do conflito
- Acordo de colaboração para documentar após a mediação
- Sinais de alerta para monitorar após a resolução
