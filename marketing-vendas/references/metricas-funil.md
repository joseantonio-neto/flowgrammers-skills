# Métricas de Funil de Vendas — Benchmarks por Indústria BR

> Última atualização: Mai/2026. Fontes: Leadster Panorama 2025, RD Station, IdeaSales, Shopify BR, Ploomes.

---

## Fórmula Base

```
Taxa de Conversão = (Número de Conversões / Total de Visitantes/Leads) × 100
```

**Tipos de conversão:**
- Visitante → Lead (topo de funil)
- Lead → MQL (qualificação)
- MQL → SQL (pré-venda)
- SQL → Proposta
- Proposta → Fechamento

## Benchmarks de Funil Completo por Modelo de Negócio

```
Modelo Inbound Marketing (B2B):
Visitantes → Leads: 2-5%
Leads → MQL: 25-40%
MQL → SQL: 20-30%
SQL → Proposta: 50-70%
Proposta → Fechamento: 20-35%

Exemplo prático:
10.000 visitantes → 300 leads → 90 MQL → 27 SQL → 18 propostas → 5-6 clientes
```

## Métricas de Saúde do Funil

### Custo por Lead (CPL) por setor BR
| Setor | CPL Médio |
|---|---|
| E-commerce | R$8 – R$25 |
| SaaS SMB | R$30 – R$80 |
| Educação online | R$10 – R$40 |
| Imobiliário | R$50 – R$200 |
| Saúde / Clínicas | R$20 – R$80 |
| B2B Enterprise | R$150 – R$500+ |

### LTV vs. CAC (referência saudável)
- LTV:CAC mínimo aceitável: 3:1
- LTV:CAC saudável: 5:1
- LTV:CAC excelente: 10:1+
- Payback period saudável: < 12 meses (SaaS), < 6 meses (e-commerce)

## Sinais de Alerta no Funil

| Sintoma | Problema Provável | Ação |
|---|---|---|
| Alto tráfego, baixo lead | Landing page ruim / oferta fraca | Revise CTA e proposta de valor |
| Alto lead, baixo MQL | Leads não qualificados | Revise critérios de segmentação |
| Alto MQL, baixo SQL | Qualificação de pré-venda fraca | Treine SDR / melhore processo de descoberta |
| Alto SQL, baixo fechamento | Proposta ou pricing inadequados | Revise proposta de valor e precificação |
| Alto churn pós-venda | Produto não entrega o prometido | Alinhamento entre vendas e produto |
