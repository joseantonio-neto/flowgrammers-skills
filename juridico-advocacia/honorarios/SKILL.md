# Honorários Advocatícios

Precificação, tabelas de referência e estruturação de honorários para advogados e escritórios no Brasil, considerando as tabelas das seccionais da OAB.

## Quando Usar
- Definir honorários para um novo cliente ou caso
- Criar tabela de serviços para o escritório
- Estruturar contrato de honorários de sucumbência

## Exemplos de Prompt
```
Sou advogado trabalhista com 5 anos de experiência em SP e quero criar uma
tabela de honorários para serviços recorrentes. Inclua: consulta inicial,
acompanhamento de processo, elaboração de contratos e pareceres.
Compare com a tabela de referência da OAB-SP.
```

```
Fui contratado para acompanhar uma ação de cobrança de R$ 200.000 em SP.
Quais os modelos de honorários possíveis? (fixo, êxito, misto)
Crie uma proposta de honorários com 3 opções para apresentar ao cliente.
```

## Saídas Típicas
- Tabela de serviços com faixas de honorários
- Comparação com tabelas de referência OAB por seccional
- Modelos de contrato de honorários (fixo, êxito, misto)
- Cálculo de honorários de sucumbência (art. 85 CPC/2015)
- Dicas de negociação e apresentação de proposta
