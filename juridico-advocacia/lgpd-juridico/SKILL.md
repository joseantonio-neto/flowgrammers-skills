# LGPD para Escritórios Jurídicos

Adequação à Lei Geral de Proteção de Dados (Lei 13.709/2018) para escritórios de advocacia: inventário de dados, cláusulas contratuais, políticas e procedimentos internos.

## Quando Usar
- Adequar escritório à LGPD
- Incluir cláusulas de LGPD em contratos com clientes
- Criar política de privacidade para site do escritório

## Exemplos de Prompt
```
Meu escritório de advocacia tem dados de 300+ clientes em planilhas e emails.
Crie um plano de adequação à LGPD em 4 etapas com: mapeamento de dados,
medidas de segurança, cláusulas contratuais e procedimento de resposta
a solicitações de titulares.
```

```
Preciso de uma cláusula de tratamento de dados pessoais para incluir
no meu contrato de prestação de serviços jurídicos. A cláusula deve cobrir:
base legal, finalidade, prazo de retenção e direitos do titular.
```

## Saídas Típicas
- Inventário de dados (RoPA simplificado para escritório)
- Cláusulas de LGPD para contratos de honorários
- Política de privacidade para site do escritório
- Procedimento de atendimento a titulares (acesso, exclusão, correção)
- Checklist de conformidade básica para escritórios
