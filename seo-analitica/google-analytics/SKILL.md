# Google Analytics 4 (GA4)

Configuração avançada, implementação de eventos e análise de dados no GA4 para e-commerces, SaaS e sites de conteúdo brasileiros.

## Quando Usar
- Configurar GA4 corretamente em um site ou app
- Implementar eventos customizados para rastrear conversões
- Analisar dados e extrair insights de negócio do GA4

## Exemplos de Prompt
```
Tenho um SaaS B2B e quero rastrear a jornada completa do usuário no GA4.
Liste os 15 eventos customizados mais importantes para SaaS, com: nome do evento
em formato GA4 (snake_case), parâmetros a enviar, código gtag.js de implementação
e qual insight de negócio cada evento gera.
```

```
Meu GA4 está configurado mas o relatório de funil de compra não faz sentido:
mostra 0 compras mas meu Shopify registra 45 pedidos na semana. 
Quais as causas mais comuns deste problema e como debugar usando o DebugView?
```

## Saídas Típicas
- Lista de eventos customizados com código de implementação
- Guia de configuração de conversões no GA4
- Relatórios recomendados por tipo de negócio
- Procedimento de debugging com DebugView
- Configuração de audiências para uso em Google Ads
