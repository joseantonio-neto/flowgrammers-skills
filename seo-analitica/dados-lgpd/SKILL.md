# Dados e LGPD

Estratégias de coleta, armazenamento e uso de dados de marketing digital em conformidade com a Lei Geral de Proteção de Dados (LGPD, Lei 13.709/2018).

## Quando Usar
- Adequar estratégia de dados ao compliance da LGPD
- Implementar banner de cookies conforme a lei
- Criar política de privacidade para site ou app

## Exemplos de Prompt
```
Meu site usa GA4, Meta Pixel, HotJar e captura emails via formulário.
Crie um plano de adequação completo à LGPD com: inventário de dados coletados,
base legal para cada coleta, banner de cookies com categorias e consentimento
granular, política de privacidade e como responder a requisições de titulares.
```

```
Quero criar uma estratégia de email marketing para um e-commerce com
LGPD em mente. Quais as bases legais permitidas para envio de email?
Como coletar consentimento corretamente? Como fazer o opt-out e armazenar
os comprovantes de consentimento?
```

## Saídas Típicas
- Inventário de dados coletados com base legal para cada um
- Especificação de banner de cookies com categorias obrigatórias
- Template de política de privacidade adaptado ao negócio
- Processo de atendimento a requisições de titulares (acesso, exclusão, portabilidade)
- Checklist de compliance para marketing digital conforme LGPD
