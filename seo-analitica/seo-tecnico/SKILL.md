# SEO Técnico

Auditoria e otimização técnica de SEO: Core Web Vitals, rastreamento, indexação, schema markup e correções prioritárias para sites brasileiros.

## Quando Usar
- Auditar problemas técnicos de SEO em um site
- Priorizar correções técnicas por impacto
- Implementar schema markup para rich results

## Exemplos de Prompt
```
Faça um checklist de auditoria SEO técnica para um e-commerce de roupas
com 2.000 páginas de produto no WordPress/WooCommerce. Priorize por:
impacto no ranking, velocidade de implementação e recursos necessários.
Inclua: Core Web Vitals, indexação, URLs, meta tags, schema e links internos.
```

```
Meu site tem LCP de 6.2 segundos no mobile. Quais são as causas mais comuns
deste problema em sites WordPress e quais as soluções técnicas em ordem de
impacto, do mais fácil ao mais complexo?
```

## Saídas Típicas
- Checklist de auditoria SEO técnica priorizada
- Plano de ação por nível de complexidade (rápido/médio/complexo)
- Snippets de schema markup para o tipo de site
- Guia de Core Web Vitals com limites e como medir
- Relatório de auditoria em formato executivo e técnico
