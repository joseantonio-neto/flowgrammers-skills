# Relatório de Dados

Estruturação e criação de relatórios executivos e operacionais de marketing digital: do template de apresentação à análise de dados para tomada de decisão.

## Quando Usar
- Criar relatório mensal de marketing para diretoria
- Estruturar dashboard de acompanhamento de KPIs
- Apresentar resultados de campanha para cliente

## Exemplos de Prompt
```
Preciso criar um relatório mensal de marketing digital para apresentar
para o CEO de uma PME. Fontes: GA4, Meta Ads, Google Ads e RD Station.
Crie a estrutura completa com: resumo executivo (1 página), análise por canal,
highlights e lowlights do mês, e plano para o próximo mês.
```

```
Sou gestor de tráfego e preciso de um template de relatório de campanha
para enviar para cliente após cada mês. Formato: PDF de 6-8 páginas.
Inclua: capa, resumo de performance, análise por campanha, criativos que
performaram melhor e próximos testes recomendados.
```

## Saídas Típicas
- Template de relatório por audiência (CEO/diretoria/cliente/operacional)
- Estrutura de resumo executivo de 1 página
- Análise de cada canal com métricas e interpretação
- Visualizações recomendadas para cada tipo de dado
- Formato de apresentação de recomendações e próximos passos
