# Google Search Console

Uso avançado do GSC para diagnosticar problemas de SEO, identificar oportunidades de conteúdo e melhorar rankings no Brasil.

## Quando Usar
- Diagnosticar queda de tráfego orgânico
- Encontrar oportunidades de melhoria de CTR e posição
- Verificar problemas de indexação e cobertura

## Exemplos de Prompt
```
Meu site teve uma queda de 40% no tráfego orgânico nos últimos 30 dias.
Crie um passo a passo de diagnóstico no Google Search Console para identificar:
se é uma penalidade, perda de indexação, mudança de algoritmo ou sazonalidade.
Inclua quais relatórios verificar e o que procurar em cada um.
```

```
Tenho 200 URLs com problema de indexação no GSC. Quais as causas mais comuns
de "Excluídas pelo tag noindex", "Rastreadas mas não indexadas" e
"URL duplicada, o Google escolheu uma diferente"? E como priorizar correções?
```

## Saídas Típicas
- Diagnóstico de queda de tráfego passo a passo
- Interpretação dos relatórios de Cobertura, Desempenho e Melhorias
- Plano de ação para correção de problemas de indexação
- Estratégia para melhorar CTR nas SERPs
- Monitoramento mensal de métricas-chave do GSC
