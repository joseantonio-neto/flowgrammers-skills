# Testes A/B

Planejamento, execução e análise estatística de testes A/B e multivariados para otimização de conversão em sites, apps e emails brasileiros.

## Quando Usar
- Planejar teste A/B para uma página ou elemento
- Calcular tamanho de amostra necessário
- Analisar resultados e declarar vencedor com confiança

## Exemplos de Prompt
```
Quero testar dois CTAs na minha landing page de SaaS: "Começar grátis" vs "Ver demonstração".
A página atual tem 1.200 visitas/mês e taxa de conversão de 3.2%. Quantos visitantes
preciso em cada variante para ter 95% de significância? Quanto tempo o teste vai durar
e como evitar declarar vencedor cedo demais?
```

```
Rodei um teste A/B no meu e-mail de abandono de carrinho por 3 semanas.
Versão A: 1.245 envios, 156 cliques (12.5% CTR). Versão B: 1.238 envios, 189 cliques (15.3% CTR).
O resultado é estatisticamente significativo? Como calcular e quando posso
implementar a versão vencedora com segurança?
```

## Saídas Típicas
- Calculadora de tamanho de amostra e duração do teste
- Guia de priorização de testes (ICE Score)
- Análise de significância estatística com interpretação
- Template de documentação de hipótese de teste
- Checklist de erros comuns em testes A/B
