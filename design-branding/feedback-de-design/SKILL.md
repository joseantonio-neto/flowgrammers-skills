# UI Feedback

Revisão estruturada, feedback construtivo e direção de arte para projetos de interface digital: apps, sites, dashboards e protótipos.

## Quando Usar
- Revisar protótipo ou tela antes de passar para desenvolvimento
- Dar feedback estruturado para designer ou agência
- Avaliar usabilidade de uma interface existente

## Exemplos de Prompt
```
Tenho um protótipo de app de controle financeiro pessoal. Preciso de um framework
de feedback de UI/UX para revisar com meu designer. Crie um checklist de 20 pontos
cobrindo: hierarquia visual, usabilidade, consistência e acessibilidade.
```

```
Nossa tela de checkout tem abandono de 68%. Com base nas melhores práticas de UX,
quais são os 5 problemas mais comuns em checkouts e como priorizar correções
para impactar mais rapidamente a conversão?
```

## Saídas Típicas
- Checklist estruturado de revisão de UI/UX
- Feedback organizado por prioridade (crítico/importante/sugestão)
- Referências de melhores práticas do mercado
- Próximos passos e critérios de aprovação
- Template de sessão de design review
