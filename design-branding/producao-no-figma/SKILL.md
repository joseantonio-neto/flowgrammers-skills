# Design no Figma

Boas práticas de organização, componentização e produção em Figma para times de design e desenvolvedores no Brasil.

## Quando Usar
- Organizar arquivos Figma para trabalho em equipe
- Criar ou revisar sistema de design no Figma
- Orientar desenvolvedor sobre como extrair assets do Figma

## Exemplos de Prompt
```
Nosso arquivo Figma está desorganizado com 8 designers trabalhando nele.
Crie uma estrutura de organização de arquivo Figma com: nomenclatura de pages,
naming conventions de layers/frames, estrutura de componentes e documentação
de como usar cada componente.
```

```
Preciso criar um design system no Figma para nosso produto SaaS.
Por onde começo? Crie um plano de 4 semanas para construir do zero:
tokens de design, componentes básicos, layouts e documentação.
```

## Saídas Típicas
- Estrutura de organização de arquivo Figma
- Naming conventions para layers, componentes e variants
- Checklist de boas práticas de componentização
- Guia de auto-layout e como aplicar
- Template de documentação de componentes
