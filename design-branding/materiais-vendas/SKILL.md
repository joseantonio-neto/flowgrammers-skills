# Materiais de Vendas

Criação e estruturação de materiais comerciais visuais para ciclos de venda B2B e B2C no mercado brasileiro: one-pagers, propostas, folders e kits comerciais.

## Quando Usar
- Criar one-pager para envio após reunião de vendas
- Estruturar proposta visual para cliente corporativo
- Desenvolver folder ou brochure para produto/serviço

## Exemplos de Prompt
```
Crie um one-pager A4 para nossa plataforma de gestão de estoque para varejo.
Público: gerentes de operações em redes com 5+ lojas.
Tom: direto, dados concretos, sem jargão técnico.
Estruture: problema, solução, como funciona, resultados e CTA.
```

```
Preciso de uma proposta visual em PDF de 5 páginas para vender um projeto
de consultoria de transformação digital para uma indústria de médio porte.
Inclua estrutura, o que vai em cada página e dicas de apresentação visual.
```

## Saídas Típicas
- Estrutura detalhada do material com seções e hierarquia
- Textos de apoio para cada seção (headline, corpo, CTA)
- Especificações técnicas (formato, dimensões, resolução)
- Dicas de dados e prova social para incluir
- Versão digital (PDF interativo) e impressa (offset)
