# Apresentação de Slides

Estrutura narrativa, design e boas práticas para apresentações executivas, comerciais e institucionais no contexto brasileiro.

## Quando Usar
- Criar apresentação para reunião com diretoria ou conselho
- Estruturar deck comercial para cliente de alto valor
- Revisar e melhorar uma apresentação existente

## Exemplos de Prompt
```
Tenho uma apresentação de resultados trimestrais para apresentar para o conselho
de administração. São 15 slides com muita informação. Reestruture para 8 slides
impactantes com foco em: resultados vs meta, destaques, desafios e próximos passos.
```

```
Preciso de um template de apresentação comercial no Google Slides para minha
equipe de vendas usar com clientes. Crie a estrutura com: capa, agenda,
slides de problema/solução, casos de sucesso, proposta de valor e próximos passos.
```

## Saídas Típicas
- Estrutura narrativa da apresentação (storyboard)
- Recomendações de design por tipo de slide
- Regras de simplicidade visual (máximo por slide)
- Template de slide mestre com variações
- Guia de apresentação oral para cada seção
