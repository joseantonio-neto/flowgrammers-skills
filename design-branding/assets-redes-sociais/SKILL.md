# Social Assets

Criação de especificações, templates e sistemas visuais para redes sociais brasileiras. Padronização visual para Instagram, LinkedIn, TikTok e WhatsApp.

## Quando Usar
- Criar sistema de templates para perfil profissional ou empresa
- Especificar formatos e dimensões para equipe de design
- Padronizar identidade visual nas redes sociais

## Exemplos de Prompt
```
Crie um sistema completo de templates para Instagram de uma agência de marketing digital.
Inclua: post feed 1:1, carrossel (6 slides), stories 9:16 e capa de destaque.
Para cada: dimensões, zonas seguras, hierarquia visual e elementos fixos vs variáveis.
```

```
Preciso de especificações de banner para LinkedIn para um escritório de contabilidade.
Formatos: banner de perfil pessoal, banner de página empresa e post nativo.
Tom: sóbrio e profissional. Inclua guia de adaptação de conteúdo para cada formato.
```

## Saídas Típicas
- Especificações técnicas por formato e plataforma
- Sistema de grid e zonas de segurança
- Hierarquia visual (o que muda e o que é fixo)
- Paleta de cores e tipografia adaptada por rede social
- Guia de adaptação de conteúdo entre formatos
