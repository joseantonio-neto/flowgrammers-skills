# Brand Book

Estruturação e criação de brandbooks completos com todas as diretrizes de uso da marca: aplicações corretas, restrições, tipografia, paleta e tom de voz.

## Quando Usar
- Criar brandbook para nova identidade visual
- Documentar e padronizar uso da marca para equipe interna
- Preparar material de onboarding para agências e parceiros

## Exemplos de Prompt
```
Nossa empresa tem logo, paleta e tipografia definidos mas nunca documentamos.
Crie a estrutura completa de um brandbook com: logo (usos corretos, incorretos, variações),
paleta primária/secundária/neutra com HEX/RGB/CMYK, tipografia com exemplos de uso,
elementos gráficos e tom de voz em 3 palavras + exemplos.
```

```
Preciso de um brandbook simplificado (brand kit) de 8 páginas para enviar
para influenciadores e parceiros que vão usar nossa marca.
O que incluir e como torná-lo fácil de usar por quem não é designer?
```

## Saídas Típicas
- Estrutura completa do brandbook por seção
- Exemplos de aplicações corretas e incorretas do logo
- Paleta documentada com códigos técnicos completos
- Diretrizes tipográficas com hierarquia e pesos
- Seção de tom de voz e linguagem da marca
