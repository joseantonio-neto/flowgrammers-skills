# Pitch Design

Estrutura narrativa, hierarquia visual e design de pitch decks para investidores, clientes corporativos e apresentações executivas no contexto brasileiro.

## Quando Usar
- Criar ou revisar pitch deck para investidores (Seed, Série A, B)
- Estruturar apresentação comercial para clientes enterprise
- Preparar deck para eventos e aceleradoras

## Exemplos de Prompt
```
Crie a estrutura completa de um pitch deck de 12 slides para captação de R$ 3M
em Série A para uma startup de agtech. Diga o que vai em cada slide,
a hierarquia visual recomendada e os dados que preciso ter prontos.
```

```
Tenho um deck de vendas B2B para SaaS de RH com 25 slides desorganizados.
Reestruture para uma apresentação de 10 slides para decisor (VP de RH).
Foco: problema, solução, ROI e próximos passos.
```

## Saídas Típicas
- Estrutura slide-a-slide com objetivo de cada tela
- Hierarquia visual (o que é H1, H2, destaque)
- Dados e prova social necessários por slide
- Dicas de design para cada tipo de slide
- Versão longa (pitch completo) e curta (elevator pitch visual)
